The Charity theme is based on the Classic Blog Theme Pack's “Classic Blog, Minimalist” theme. In addition to the capabilities of that theme, Charity introduces some additional features and flexibility aimed at making it easy to start a charity-centric web site. Some of the features of this theme:

  • Several colors and layout options to choose from.
  • Include an image in the header of the site.
  • Include a navigation menu based on Page content.
  • Include an image gallery with any (or every) Page and Entry.
  • Support for Action Streams to collate activity on other web sites.
  • Customize the Facebook Like button design.

Two and three column layouts are supported. In the Page Layout Theme Option, “t” stands for “thin,” “w” stands for “wide,” and “m” is a “minor” column though wider than “thin.” Republish the site after changing layouts.

Manage the sidebars of this blog with Widgets and Widget Sets. In the Design menu you can visit Widgets, then select a Widget Set and edit its contents with drag-n-drop simplicity. Use any of the available Widgets to customize this blog, or create new Widgets to add to the sidebar.

Theme Options

Visit Theme Options (in the Design menu) to set options for this blog.

The Facebook Like Button and PayPal Donations options rely upon those Widgets being a part of the Widget Sets used for the site. (By default, these Widgets are included and are therefore functional.)

Pages and Entries

Entries are used for the actual “blog” component of this site. Pages are used as other unique types of content for the site. A handful of Pages have been created to help you get started.

Both Entries and Pages support image galleries. To see the Image Gallery field, visit the Display Options in the upper-right and enable the Image Gallery field. Select files to upload, then save. After the Page or Entry is saved, the label and description of each image in the gallery may be edited (though it is not necessary to change them).

Publishing

Some Templates use the Publish Queue to publish less-frequently visited archives in the background, making this blog feel faster. Be sure that run-periodic-tasks has been properly set up with this Melody or Movable Type installation so that the Publish Queue can run.

Server Side Includes (SSI) are enabled on this theme, a feature which can further improve publishing times. Your server will need to be able to parse .html files as PHP. To disable SSI, visit the Preferences menu and choose Publishing. Be sure to republish after setting or changing the SSI option. Configuring SSI is beyond the scope of this document.
